Topology Notes

PP-01
- Serves as the central termination point for horizontal cabling runs from office endpoints.
- Utilizes punchdown terminations on the rear of the panel and RJ45 jack ports on the front.
- Provides organized cable management and simplifies moves, adds, and changes within the network.
- Connected to the access switch using patch cables.

ACCESS-SW1
- Provides network connectivity for endpoint devices including workstations, printers, and wireless access points.
- Serves as the primary access layer switch within the network.
- Connected to the patch panel to support structured cabling practices, as well as the distribution switch.

CORE-SW1
- Aggregates connections from the access layer.
- Serves as the central network infrastructure device within the topology.
- Provides backbone connectivity between network devices and services.
- Connected to the router through a fiber uplink utilizing SFP transceivers.

RTR-01
- Represents the network edge and external connectivity.
- Connected to the core switch through a dedicated fiber uplink.
- Demonstrates the role of routing devices within an enterprise environment.

DNS-SRV1
- Represents infrastructure services commonly found within enterprise networks.
- Demonstrates server integration within a rack-based network design.
- Connected directly to network switching infrastructure.

FILE-SRV1
- Represents enterprise file storage and shared resource services.
- Demonstrates server integration within a rack-based network design.
- Connected directly to network switching infrastructure.
